About the Project

This page contains information about our NPC generator.

How it works

NPC Generator API is an innovative tool based on Gemini artificial intelligence, created to generate and manage rich, complex non-player characters (NPCs) based on provided narratives and world histories.

Main Features

  • World Creation - adding extensive histories that form the foundation for characters
  • Character Generation - creating NPCs with unique personality traits, faction affiliations, and backstories
  • Interactive Conversations - ability to conduct dialogues with generated characters
  • Context Memory - a system that remembers conversation history with characters

How It Works

  1. The application uses the powerful Gemini AI model to analyze provided world histories
  2. Based on the world context, coherent characters with unique personalities are generated
  3. Each character responds individually to user questions and requests, according to their personality and background
  4. Conversation history is saved, allowing for building long-term relationships with characters
Technologies
  • Backend: Django 5.2 with Django REST Framework
  • AI Engine: Google Gemini
  • Containerization: Docker
  • User Interface: HTML/CSS/JavaScript
